# Telecom Customer Churn Analysis Using Excel

## Project Overview

Analyzed telecom customer data to identify key drivers of customer churn using Excel Pivot Tables, KPI calculations, and dashboard design.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Data Visualization
- Dashboard Design

## Skills Demonstrated

- Data Cleaning
- Data Analysis
- KPI Development
- Dashboard Design
- Business Insights

## Key Findings

- Overall churn rate: 14.07%
- Customers with International Plans show a churn rate of 42.17%.
- Customers making 4–5 customer service calls show a churn rate of 48.62%.
- Churn increases as customer-service interactions increase.

## Recommendations

- Review International Plan offerings.
- Improve first-contact issue resolution.
- Monitor customers with repeated service interactions.
- Launch retention campaigns for high-risk customers.
